// voice control

Talk to your canvas.

A particle VoiceOrb listens, thinks, and speaks — editing the graph as you talk. Ask it to add a node, swap a model, or run the chain. It responds on the canvas.

transcript · idle
""
Investigate
Review

The VoiceOrb visualizes four states: idle (a gentle breathing sphere), listening (the surface ripples toward your cursor as it captures speech), thinking (turbulent high-frequency motion while it plans the graph edit), and speaking (a pulsing amplitude envelope as it narrates the change).

// inbuilt terminal

A real terminal, inside the canvas.

Run the graph, stop it, retry a single node, stream logs — without leaving the app.

dispatch — run
// commands
  • run — execute the whole chain
  • stop — halt the active run
  • retry <node> — rerun one node
  • clear — wipe the log
  • help — list commands
