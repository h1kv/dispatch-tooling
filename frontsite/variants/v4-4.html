// agentic handoff

Context and control, passed downstream.

Each node hands its output — and the run itself — to the next, even across providers. The baton carries context from OpenAI to Anthropic to Google without glue code.

// inbuilt terminal

A real terminal, inside the app.

Run, stop, retry a node, clear, help — streamed logs as the chain executes.

dispatch — run
// command reference
  • run — execute the chain
  • stop — halt the run
  • retry <node> — re-run one node
  • clear — clear the log
  • help — list commands
