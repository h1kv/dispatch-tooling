Model hotswapping

Pick the model per node. Swap live.

Choose the provider and the exact model on any node, then change it mid-build. No lock-in, no redeploy — the next run uses your pick.

  • OpenAI — gpt-4o · gpt-4o-mini · o3 · o4-mini
  • Anthropic — claude-opus-4-8 · claude-sonnet-4-6 · claude-haiku-4-5
  • Google — gemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro

How it runs

Place. Wire. Run.

01

Place nodes

Drop node types onto the infinite dotted canvas.

02

Wire top-to-bottom

Connect with bezier edges; output flows node→node.

03

Press Run

The chain executes; Review pauses for Approve/Reject.

04

Branch & merge

Parallel fans out, Merge recombines, Deploy ships.
