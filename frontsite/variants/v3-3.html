How it runs

Place. Wire. Run.

Drop nodes on the canvas, connect them top-to-bottom, and press Run. Output flows node to node. Review nodes pause for a human; Parallel and Merge handle branching.

  1. 01
    Place nodes

    Pull typed nodes from the toolbox onto the infinite dotted canvas.

  2. 02
    Wire top-to-bottom

    Connect outputs to inputs with clean bezier edges in each node's accent.

  3. 03
    Press Run

    The sequencer walks the graph — status cycles idle → running → done, edges flow amber.

  4. 04
    Approve & ship

    A Review node pauses for sign-off; Deploy ships the result. Retry any node from the CLI.
