// real-time execution

Watch the run, edge by edge.

Nodes show idle, running, done, paused and error states as they execute. Edges animate while data is in flight, so you always see exactly where the chain is.

  • Multi-provider per node — OpenAI, Anthropic, or Google.
  • Live tools: web_search, fetch_url, read_file, write_file, list_files, shell_exec.
  • Review nodes pause the run for human approval.
  • Condition branching and Memory shared state across the graph.
