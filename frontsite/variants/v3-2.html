Model hotswapping

Pick the provider and model per node. Swap live.

No lock-in. Run Investigate on a fast model, Plan on a frontier reasoner, Create on whatever ships best. Change it mid-run — the next node picks up the new model instantly.

How it runs

Place. Wire. Run.

STEP 01

Place nodes

Drag typed nodes onto the infinite dotted canvas.

STEP 02

Wire top-to-bottom

Connect outputs to inputs — bezier edges carry data downstream.

STEP 03

Press Run

The chain executes node→node, each passing output to the next.

STEP 04

Pause & branch

Review nodes wait for a human; Parallel/Merge fan out and back in.
