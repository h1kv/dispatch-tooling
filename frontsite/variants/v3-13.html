MODEL HOTSWAPPING

Pick the model per node. Swap it live.

Every node owns its provider and model. Click a provider, pick a model — the node updates instantly with an amber confirm pulse. No lock-in, no redeploy.

  • OpenAI gpt-4o · gpt-4o-mini · o3 · o4-mini
  • Anthropic claude-opus-4-8 · claude-sonnet-4-6 · claude-haiku-4-5
  • Google gemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro

HOW IT RUNS

Place, wire, run.

  1. 01

    Place nodes

    Drag node types onto the infinite canvas.

  2. 02

    Wire top-to-bottom

    Connect outputs to inputs. Each node feeds the next.

  3. 03

    Press Run

    The chain executes. Review pauses for Approve / Reject; Parallel fans out, Merge fans in.

  4. 04

    Ship

    Deploy nodes push the result. Watch it all stream live.
