// How it works

Place. Wire. Run.

01

Place nodes

Drag from the library onto an infinite canvas. Each node gets its own provider and model.

02

Wire them

Connect outputs to inputs top-to-bottom. Branch with Condition, persist with Memory.

03

Press Run

The chain executes automatically. Watch nodes flip idle → running → done as edges animate.

Build your first agent workflow today.

Clone it, run it locally, own every node. No accounts, no lock-in.
