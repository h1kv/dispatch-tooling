// How it runs

Wire it. Run it. Watch it.

Press Run and the chain executes automatically, top to bottom. Every node reports its state as it goes — and the edge connecting two nodes animates the moment data flows across it.

  • Condition branching

    Route True/False down different paths of the graph.

  • Shared Memory

    Read and write a shared store any node can reach.

  • Review checkpoints

    Pause the run for a human Approve / Reject decision.
