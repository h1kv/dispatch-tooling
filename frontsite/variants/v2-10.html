// press run

The chain runs itself.

Hit Run and DISPATCH walks the graph top-to-bottom. Edges animate as data flows, Review nodes pause for you, and Condition nodes pick a branch. Watch it in the log.

  • Output of each node feeds the next automatically.
  • Live state per node: idle, running, done, paused, error.
  • Human checkpoints hold until you approve.
