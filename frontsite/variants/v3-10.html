INBUILT CLI

A real terminal, inside the canvas.

Drive the whole graph from the keyboard. Run, stop, and retry any node by name. Logs stream in with timestamps and levels, so agents are debuggable like any other process.

run stop retry <node> clear help

HOW IT RUNS

Place. Wire. Run.

  1. 01

    Place nodes

    Drag node types from the toolbox onto the infinite dotted canvas.

  2. 02

    Wire top-to-bottom

    Connect outputs to inputs with bezier edges. Each node feeds the next.

  3. 03

    Press Run

    The chain executes in order — Review pauses for Approve/Reject, Parallel fans out.

  4. 04

    Ship

    Merge fans back in, Deploy ships the result. Watch every status live.
