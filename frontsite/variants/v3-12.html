Model hotswapping

Pick the model. Per node. Live.

Choose the provider and the exact model for every node — and swap it mid-run. No lock-in, no redeploy. Route reasoning to one model, drafting to another.

  • OpenAI gpt-4o · gpt-4o-mini · o3 · o4-mini

  • Google gemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro
Inbuilt CLI

A terminal inside the canvas.

Drive the whole graph from the keyboard. Run the chain, stop it, retry a single node, clear logs. Every execution streams timestamped, level-colored logs — fully agent-debuggable.

run stop retry <node> clear help

How it runs

Place. Wire. Run.

01

Place nodes

Drag node types onto the infinite canvas and configure each one.

02

Wire top-down

Connect outputs to inputs with bezier edges. Fan out with Parallel, fan in with Merge.

03

Press Run

The chain executes node to node, each passing output to the next. Review nodes pause for approval.

04

Ship

Watch statuses cycle, edges flow, logs stream — then Deploy to prod.
