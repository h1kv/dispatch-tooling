// NODE TYPES

Twelve real nodes.
Wire them however the work flows.

No invented primitives. Every node does one job and passes its output down the graph.

// HOW IT RUNS

Place, wire, Run.

01 / place

Drop nodes on the canvas

Pick from the 12 node types. Each carries its own provider, model, prompt, and tools.

02 / wire

Connect top-to-bottom

Edges define order. Condition nodes branch True/False; Memory holds shared state across the graph.

03 / run

Press Run, watch it execute

Nodes show idle / running / done / paused / error. Edges animate while data flows. Review nodes pause for you.
