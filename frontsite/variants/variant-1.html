Why DISPATCH.AI

Everything an agent chain needs, on one canvas.

Compose multi-step AI work the way you'd sketch it on a whiteboard — then watch it execute in real time.

Multi-provider

Any model, per node

Pick OpenAI, Anthropic, or Google — and the exact model — independently on every node in the chain.

Live tools

Agents that act

Nodes call web_search, fetch_url, read_file, shell_exec and more.

Human-in-the-loop

Pause for approval

Drop a Review node anywhere to halt the chain for a human Approve or Reject before it continues.

Branching

Route on the output

Condition nodes split the path True / False based on the previous node's result.

Shared memory

State across the graph

Memory nodes read and write a shared context store, so any node downstream can use it.

Real-time

Watch it run

Every node shows idle, running, done, paused, or error — and edges animate as data flows through.

How it works

Three steps from idea to running chain.

01

Drop your nodes

Start with a Start node, then add Investigate, Plan, Design, Create, Evaluate, Document — whatever the job needs.

02

Wire them top-to-bottom

Connect outputs to inputs. Add a Condition to branch, a Review to checkpoint, a Memory node to share state.

03

Press Run

The chain executes automatically — each node passing output to the next, live status on every step.

The node library

Twelve building blocks.

Each node is a single, well-scoped step. Combine them to model any workflow.

Start

Document

Condition

Branch True / False.

Tool Call

Memory

Shared in-chain context store.
