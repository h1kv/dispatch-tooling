Model hotswapping

Pick the provider and model per node. Swap live.

No lock-in. Route a planning node to a frontier model and a cheap loop to a mini. Click to swap — the node updates instantly.

How it runs

Place. Wire. Run.

01

Place nodes

Drop nodes from the toolbox onto the infinite canvas.

02

Wire top-to-bottom

Connect outputs to inputs. Fan out with Parallel, fan in with Merge.

03

Press Run

The chain executes node→node, each output feeding the next.

04

Review & ship

Pause at a Review node, approve, then Deploy.
