MODEL HOTSWAPPING

Any model. Per node. Swapped live.

Pick the provider and the exact model on every node — then swap it mid-run. Route reasoning to a frontier model and grunt work to a fast one. No lock-in, no redeploy.

  • OpenAIgpt-4o · gpt-4o-mini · o3 · o4-mini
  • Anthropicclaude-opus-4-8 · claude-sonnet-4-6 · claude-haiku-4-5
  • Googlegemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro

INBUILT CLI

A real terminal, inside the canvas.

Drive the whole graph from the keyboard. Run the chain, stop it, retry a single node, clear the log. Every agent action streams back as a timestamped, level-prefixed line.

run stop retry <node> clear help

Levels: › info · ⚠ warn · ✕ error · ✓ done

HOW IT RUNS

Place. Wire. Run. Ship.

01

Place nodes

Drag node types from the toolbox onto the infinite dotted canvas.

02

Wire top to bottom

Connect outputs to inputs with bezier edges. Parallel fans out, Merge fans in.

03

Press Run

Each node executes in order, passing its output to the next. Review nodes pause for Approve / Reject.

04

Ship it

A Deploy node pushes the result to prod — watched live in the terminal.
