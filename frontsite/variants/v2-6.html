// how it runs

Press Run. Watch it flow.

01
Wire top-to-bottom

Connect nodes on the canvas. Each edge carries the output above it into the node below.

02
Execute the chain

DISPATCH walks the graph in order. Conditions branch, Memory persists, Review pauses for a human.

03
See it in real time

Nodes flip idle → running → done. Active edges march. Errors surface inline, never silently.
