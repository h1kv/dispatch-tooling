MODEL HOTSWAPPING

Pick the model per node. Swap it live.

Every node carries its own provider and model. Click a provider, choose a model — the node updates instantly with an amber confirm. No vendor lock-in, ever.

INBUILT CLI

A real terminal, inside the app.

Drive the whole canvas from the keyboard. Run the chain, stop it, retry a single node, clear, or ask for help — streaming level-prefixed logs the whole way.

run stop retry <node> clear help

HOW IT RUNS

Place, wire, run.

01

Place nodes

Drop nodes from the toolbox onto the infinite dotted canvas.

02

Wire top-down

Connect outputs to inputs with clean bezier edges, top to bottom.

03

Press Run

The chain executes node→node; Review pauses for Approve/Reject.

04

Branch & merge

Parallel fans out, Merge fans in. Watch status flow in real time.
