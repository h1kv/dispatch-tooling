// Workflows

Wire it top-to-bottom.
Press Run.

Output flows down the wires. Nodes light up as they execute — idle, running, done — and edges animate while data moves. Hit a Review node and the run pauses for you.

  • Real-time status on every node, every edge.
  • Condition branches and shared Memory across the graph.
  • Human Review checkpoints — Approve or Reject inline.
