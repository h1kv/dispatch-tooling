// how it runs

Wire it once. Watch it flow.

01

Output flows downstream

Connect a node's output to the next input. DISPATCH carries the payload along the edge — no glue code.

02

Branch & remember

Condition nodes split the path on True/False. A shared Memory store keeps state across the whole graph.

03

Pause for a human

Drop a Review node anywhere. The run holds — amber — until someone approves or rejects.
