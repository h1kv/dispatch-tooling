MODEL HOTSWAPPING

Pick the model per node. Swap live.

Every node carries its own provider and model. Route reasoning to one engine, drafting to another, and switch mid-run without rewiring. No lock-in, no redeploy.

  • OpenAIgpt-4o · gpt-4o-mini · o3 · o4-mini
  • Anthropicclaude-opus-4-8 · claude-sonnet-4-6 · claude-haiku-4-5
  • Googlegemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro

INBUILT CLI

A real terminal in the canvas.

Drive the whole graph from the keyboard. Run the chain, stop it, retry a single node, clear, or ask for help — every command streams timestamped, level-coloured logs you can actually debug.

run stop retry <node> clear help
dispatch — terminal

HOW IT RUNS

Place. Wire. Run.

01

Place nodes

Drag typed nodes onto the infinite canvas.

02

Wire top-to-bottom

Connect outputs to inputs with bezier edges.

03

Press Run

The chain executes node → node, output feeding forward.

04

Pause & branch

Review pauses for approval; Parallel/Merge split and rejoin.
