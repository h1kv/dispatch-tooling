Branching

Route on the result.

A Condition node reads the previous output and sends the chain down a True or False path.

Shared state

Memory across the graph.

Memory nodes read and write a shared store, so context flows between branches — not just node to node.

The node library

Twelve node types. One direction: down.

Drop a node, wire it to the one above, and it inherits that node's output as input. No glue code.

Entry point. Sets the task and default model.

Human-in-the-loop

Press Run. Step in when it matters.

Chains run on their own, but a Review node pauses everything for your approval before the next step. Reject and the branch stops; approve and the output flows on. You stay in control without babysitting every node.
