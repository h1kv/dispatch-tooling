// Node types

Twelve real nodes.

From a Start trigger to a human Review checkpoint — wire any of them, in any order, on one canvas.

// Execution

Real-time, top-to-bottom.

Press Run and the chain executes automatically. Each node passes its output to the next; edges animate while they're live.

// Control

Branch, store, approve.

Condition nodes split True/False, Memory shares state across the graph, and Review nodes pause for a human Approve or Reject.

// Tools

Reach the real world.

Nodes call live tools — web_search, fetch_url, read_file, write_file, list_files, shell_exec — against your data and disk.
