Model hotswapping

Swap the model behind any node. Live.

Every node owns its provider and model. Pick OpenAI, Anthropic, or Google — change it mid-run and the canvas reacts instantly. No lock-in, no re-wiring.

  • 01OpenAI — gpt-4o, gpt-4o-mini, o3, o4-mini
  • 02Anthropic — claude-opus-4-8, claude-sonnet-4-6, claude-haiku-4-5
  • 03Google — gemini-2.5-pro, gemini-2.0-flash, gemini-1.5-pro

↳ Try the picker — the node's model label cross-fades with an amber confirm pulse.

How it runs

Place. Wire. Run.

  1. 01

    Place nodes on the canvas

    Drag any of the 13 types onto the infinite dotted grid.

  2. 02

    Wire them top-to-bottom

    Bezier edges carry each node's output into the next. Source color = edge color.

  3. 03

    Press Run — the chain executes

    Each node runs in order, amber while live, green when done. Parallel fans out, Merge fans in.

  4. 04

    Pause at human checkpoints

    A Review node halts the run for Approve / Reject before continuing downstream.
