[ 02 ] MODEL HOTSWAPPING

Pick the provider and model — per node.

No lock-in. Each node carries its own provider and model, swappable live while the graph is open. Route reasoning to o3, drafting to gpt-4o-mini, polish to claude-opus-4-8 — your call, per step.

  • OpenAI — gpt-4o · gpt-4o-mini · o3 · o4-mini
  • Anthropic — claude-opus-4-8 · claude-sonnet-4-6 · claude-haiku-4-5
  • Google — gemini-2.5-pro · gemini-2.0-flash · gemini-1.5-pro
[ 03 ] INBUILT CLI

A real terminal inside the canvas.

Drive the whole graph from the keyboard. Run the chain, stop it mid-flight, retry a single node, clear logs, ask for help — every action streams timestamped, level-colored output you can actually debug.

run stop retry <node> clear help
dispatch — cli
[ 04 ] AGENTIC VOICE CONTROL

Talk to your canvas.

The VoiceOrb listens, thinks, speaks — and edits the graph by voice. Add a node, rewire a branch, retry a step, all hands-free.
