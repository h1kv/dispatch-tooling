// How it runs

Wire it. Run it. Watch it.

01

Place & wire

Drag nodes onto an infinite canvas and connect them top-to-bottom. The graph is the program.

02

Press Run

The chain executes automatically. Each node passes its output to the next; Condition branches, Memory persists.

03

Watch state

Nodes show idle, running, done, paused or error in real time. Edges animate while the run is live.
